## 확인 내용

- `/v1/dashboard/summary` 기존 응답과 테스트 기대값을 유지했다.
- `/v1/dashboard/overview`, `/targets`, `/coverage`, `/collection-activity`, `/realtime-heatmap`, `/storage-breakdown`, `/operations-trend`, `/missing-ranges`, `/audit-log-summary`를 추가했다.
- pagination 대상 API는 `limit`, `offset`, `total`을 반환하고 invalid query는 422를 반환한다.
- `GOODMONEYING_DASHBOARD_REFRESH_CONFIG` 파일 경로 override, 일부 키 누락 fallback, 잘못된 값 실패를 API 테스트로 검증했다.
- 저장소(repository) 패널별 메서드가 기존 summary source data와 일치함을 검증했다.
- seeded FastAPI `TestClient` 기준 새 패널 API는 warmup 후 3회 호출 모두 3초 이내에 200 OK로 응답했다.
