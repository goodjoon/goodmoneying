# 운영계 prod-home CI/CD 구현 계획(Implementation Plan)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** `release` 브랜치 push로 private GHCR(GitHub Container Registry)에 이미지를 만들고 `prod-home` 운영계 홈 인프라에 자동 배포, 검증, 롤백할 수 있게 만든다.

**Architecture:** GitHub Actions는 Mac Mini M4 organization self-hosted runner에서 실행된다. 배포 대상은 `prod-home` 프로필(profile)로 고정하고, 배포 스크립트는 프로필 설정을 읽어 Mac Mini M4의 `postgres`, APP SERVER 01의 `api/worker`, bmax-ubuntu의 `web`을 서버별 compose로 갱신한다. 운영 접근은 Tailscale 내부망 전용이며, Slack 연동은 `workflow_dispatch` 진입점만 남기고 후속 Task로 분리한다.

**Tech Stack:** GitHub Actions, Docker Compose, private GHCR, Bash, Python pytest, Playwright, uv, npm, FastAPI, React, PostgreSQL

---

## 파일 구조

### 생성할 파일

- `deploy/scripts/healthcheck-profile.sh`: profile 기준으로 API, web, worker, PostgreSQL healthcheck를 실행한다.
- `tests/scripts/test_deploy_profile.py`: 배포 프로필과 배포 스크립트의 dry-run, validation을 검증한다.

```bash
#!/usr/bin/env bash
set -euo pipefail

ROOT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")/../.." && pwd)"
PROFILE="${1:-}"
IMAGE_TAG="${2:-}"
DRY_RUN="${GOODMONEYING_DEPLOY_DRY_RUN:-0}"

fail() {
  printf '오류: %s\n' "$*" >&2
  exit 1
}

if [[ -z "$PROFILE" || -z "$IMAGE_TAG" ]]; then
  fail "사용법: deploy/scripts/deploy-profile.sh prod-home release-{short-sha}"
fi

if [[ "$PROFILE" != "prod-home" ]]; then
  fail "지원하지 않는 배포 프로필입니다: $PROFILE"
fi

PROFILE_DIR="$ROOT_DIR/deploy/profiles/$PROFILE"
source "$PROFILE_DIR/profile.env"
source "$PROFILE_DIR/hosts.env"

if [[ "$DRY_RUN" == "1" ]]; then
  printf 'profile=%s\n' "$GOODMONEYING_DEPLOY_PROFILE"
  printf 'tag=%s\n' "$IMAGE_TAG"
  printf 'infra host=%s compose=%s\n' "$GOODMONEYING_INFRA_HOST" "$GOODMONEYING_INFRA_COMPOSE"
  printf 'app host=%s compose=%s\n' "$GOODMONEYING_APP_HOST" "$GOODMONEYING_APP_COMPOSE"
  printf 'web host=%s compose=%s\n' "$GOODMONEYING_WEB_HOST" "$GOODMONEYING_WEB_COMPOSE"
  exit 0
fi

printf 'prod-home 배포를 시작합니다. tag=%s\n' "$IMAGE_TAG"
```
